ODrive Setup Instructions
To get started, run this command in your terminal: 
sudo bash -c "curl https://cdn.odriverobotics.com/files/odrive-udev-rules.rules > /etc/udev/rules.d/91-odrive.rules && udevadm control --reload-rules && udevadm trigger"
Supported Devices
- ODrive Pro
- ODrive S1
- ODrive Micro
USB Isolation
Use a USB isolator to prevent ground loops. These can damage electronics.
Connection Troubleshooting
- Make sure no other tab is using the ODrive
- GUI is not compatible with versions > 3.6
- Check cable or USB power
